# Spring-Pet-Clinic
Backend for a basic veterinarian clinic website. Built in the "Spring Framework 5: Beginner to Guru" course.

**Technologies used:**
- Spring 5
- Spring MVC
- Spring Data JPA
- Spring Boot
- Thymeleaf
- Project Lombok
- JUnit 5
- Mockito
- Circle CI
- Maven

Frontend was built by the Pivotal Spring-Pet-Clinic project.
